# LifeChronicle — Project Environment

## Overview

**LifeChronicle** is a private life-archive iOS app built with SwiftUI, designed around the **Old Money** aesthetic — understated luxury inspired by Prada's cool silver-white palette, serif typography, and minimal ornamentation. No shadows, no rounded corners, no noise. Just whitespace, 1px borders, and a single gold accent.

- **Bundle ID**: `com.lifechronicle.app`
- **Deployment Target**: iOS 17.0+
- **Swift Version**: 5.9+ (Xcode 15+)
- **Device**: iPhone only, portrait orientation

| `moodAnxious` | #B4A7B6 | Misty purple |

### Typography

- **Titles**: System serif (New York), 28pt bold, tracking +2
- **Body**: System sans-serif, 15pt regular, line spacing +9, tracking +0.3
- **Captions**: System sans-serif, 12pt, tracking +0.3

### Spacing & Sizing

- Screen horizontal padding: 24pt
- Standard element spacing: 16pt
- Gallery home card height: 78pt
- Gallery home card gap: 12pt
- Gold accent line: 1px × 28pt (home cards)
- Mood indicator line: 1px (list left-edge)
- Button height: 44pt (HIG minimum)
- Corner radius: 2pt (sharp, minimal)
- Shadows: None (hierarchy via whitespace and borders)

### Animation

- **Press effects**: `scaleEffect(0.98)` + `.spring(response:0.35, damping:0.75)` + haptic `.light`
- **List entrance**: staggered fade + slide from left (-24pt), 0.06s per-item delay, `.spring(response:0.45, damping:0.78)`
- **Home cards entrance**: staggered fade + slide up, 3× stagger delay per card
- **Save success**: checkmark scale transition + gold border flash (0.25s on, 0.5s off) + haptic `.success`
- **Philosophy**: Old Money restraint — springs are deliberate (damping ≥ 0.75), never bouncy. Sequences breathe. Nothing pops.
- All animation constants defined in `DesignSystem` (pressSpring, entranceSpring, fadeEase, staggerDelay)

## Getting Started

### Prerequisites

- macOS with Xcode 15.0+
- iOS 17.0+ device or simulator
- Apple ID (free) for device deployment

### 1. Generate Xcode project

```bash
cd LifeChronicle
xcodegen generate
```

This reads `project.yml` and produces `LifeChronicle.xcodeproj`.

### 2. Open & Run

```bash
open LifeChronicle.xcodeproj
```

Select a simulator (iPhone 17 Pro) or connected device, then `⌘R`.

### 3. Deploy to iPhone

1. Connect iPhone via USB, trust the computer
2. Enable Developer Mode: iPhone Settings → Privacy & Security → Developer Mode → toggle on → restart
3. In Xcode: select your iPhone from the device picker
4. Signing & Capabilities → check "Automatically manage signing" → Team = your Apple ID
5. `⌘R` to build and install (registers device with Apple)
6. For standalone `.ipa` export: Product → Archive → Distribute App → Debugging → Export
7. Upload to [Diawi](https://www.diawi.com) for QR code install

> **Note**: Free Apple ID provisioning expires after 7 days. Re-install to refresh.

## Key Patterns

### Press Effect Without Consuming Tap
```swift
.onLongPressGesture(minimumDuration: .infinity, maximumDistance: .infinity,
    pressing: { isPressed = $0; if $0 { hapticTrigger += 1 } },
    perform: {})
```

### Staggered List Entrance
```swift
.onAppear {
    for i in 0..<items.count {
        DispatchQueue.main.asyncAfter(deadline: .now() + Double(i) * 0.05) {
            withAnimation(.spring(response: 0.5, dampingFraction: 0.7)) {
                visibleCount = i + 1
            }
        }
    }
}
```

### iOS 18 Availability-Safe API
```swift
// DesignSystem.swift
extension View {
    @ViewBuilder
    func zoomSource(id: String, namespace: Namespace.ID) -> some View {
        if #available(iOS 18.0, *) {
            self.matchedTransitionSource(id: id, in: namespace)
        } else { self }
    }
}
```

### Value-Based Navigation (for zoom transitions)
```swift
NavigationLink(value: "diary") { ... }
    .zoomSource(id: "diary", namespace: namespace)
// ...
.navigationDestination(for: String.self) { id in
    switch id {
    case "diary": DiaryListView().zoomDestination(sourceID: "diary", namespace: namespace)
    ...
    }
}
```

## Data Model

```
Record
├── id: UUID
├── type: RecordType (.diary | .photo | .film)
├── content: String
├── mood: Mood? (.happy | .calm | .sad | .excited | .anxious)
├── imageUrl: String?           // single image (photo)
├── images: [String]?           // multiple images (film)
├── createdAt: Date
└── modifiedAt: Date?
```

Persisted as JSON array in `Documents/records.json`. Images stored individually in `Documents/Images/`.

## Conventions

- **No third-party dependencies** — pure Apple frameworks
- **No storyboards** — SwiftUI only
- **English only** — all UI labels, comments, and documentation in English
- **DesignSystem tokens only** — never hardcode colors or fonts
- **1px borders** — `DesignSystem.border` for all card/element edges
- **No shadows** — Old Money aesthetic uses whitespace for depth
- **Sharp corners** — `Rectangle()` not `RoundedRectangle()`, corner radius always 2pt when used
